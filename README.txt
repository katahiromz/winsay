winsay -- Windows says things!

WHAT' THIS?
-----------

This Windows command line tool says things!

HOW TO BUILD
------------

To build this project, you need Visual C++ or MinGW.
On VC++, you might need ATL (Active Template Library; for <atlbase.h>).
And then use CMake.
